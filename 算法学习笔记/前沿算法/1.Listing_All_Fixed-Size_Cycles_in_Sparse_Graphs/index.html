    • 定义 4: Let be an acyclic oriented graph. Assume that is given by the adjacency lists for each vertex. The sorted degenerate adjacency list of a vertex is its adjacency list in which every vertex that is pointing towards has been deleted and which has been sorted afterwards.

    论文作者使用的邻接表可能跟我们使用的不太一样，他的邻接表把连入节点 的边 和连出节点的边 都存储在节点 对应的列表中，所以在这一步需要将连入的边去掉。而我们使用的邻接表🈯只存储了连出的边，所以可以忽略删除连入边这一项。
