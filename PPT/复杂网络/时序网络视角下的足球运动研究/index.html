    2020年3月4日

    问题来源#

    2020 ICM Problem D#

    Huskies' Teaming Strategies#

    - 我们没有选这道题，因为复杂网络的题确实很复杂 - 时序网络最近也是一个很热门的课题

    Nonlinearity + Networks: A 2020 Vision (https://arxiv.org/abs/1911.03805)

    未来复杂网络研究的四个热点#

    1. 时序网络（temporal networks）
    2. 网络上的动力学过程（dynamical processes on network）
    3. 自适应网络（adaptive network）
    4. 高阶网络（high order network）

    问题描述#

    问题一（分析）：建立网络模型，描述队员的传球行为，并分析网络模式（pattern），如团队的二元、三元配置和团队队形。

    - 还要考虑比赛中的其他结构指标和网络属性。 - 当你观察互动时，你应该探索多个尺度，例如微观(成对)到宏观(所有玩家)，以及时间，例如短(分钟到分钟)到长(整个游戏或整个赛季)。

    问题描述#

    问题二（评价）：确定反映团队合作成功程度的指标(除了分数和比赛胜利之外)，如参加过的比赛类型是否具有多样性、球员之间的配合程度和不同球员贡献的整体分布。

    - 考虑不同方面。您还可以考虑其他团队级别的过程，例如适应性、灵活性、节奏或流程。 - 考虑博弈。一个重要的点是，区分战略是普遍有效的还是专门针对对手的某些应对策略的。 - 使用您已经确定的性能指标和团队级流程来创建一个模型，该模型捕获团队工作的结构、配置和动态方面。

    问题描述#

    问题三（优化）：利用你从团队合作模式研究中获得的洞察力来告诉教练，什么样的结构性策略对哈士奇队是有效的。

    从网络分析的角度说明，下个赛季他们应该做出哪些改变来提高球队的表现。

    问题描述#

    问题四（拓展）：您能概括一下您的发现，谈谈如何设计更有效的团队吗?

    为了开发研究团队表现的通用模型，还需要了解团队合作的哪些其他方面?

    What is Network Motifs#

    Network motifs are sub-graphs that repeat themselves in a specific network or even among various networks.

    From en.wikipedia.org/wiki/Network_motif

    - 维基百科上的定义比较抽象，很难弄懂 - 我从老师之前给我的一篇文章中找了一个具体的例子
