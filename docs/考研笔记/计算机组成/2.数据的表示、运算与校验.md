## 1. 数制转换

### 1.1. 十进制转二进制

整数部分：除积其余法

$(116)_{10} = (1110100)_2$

$116 / 2 = 58 ... 0, x_0 = 0$

$58 / 2 = 29 ... 0, x_1 = 0$

$29 / 2 = 14 ... 1, x_2 = 1$

$14 / 2 = 7 ... 0, x_3 = 0$

$7 / 2 = 3 ... 1, x_4 = 1$

$3 / 2 = 1 ... 1, x_5 = 1$

$1 / 2 = 1 ... 1, x_6 = 1$

小数部分：乘基取余法

$(0.625)_{10} = (0.101)_2$

$0.625 \times 2 = 1.25, x_0 = 1$
$0.25 \times 2 = 0.5, x_1 = 0$
$0.5 \times 2 = 1, x_2 = 1$

整数表示为$x_n \cdots x_0$，小数表示为 $x_0 \cdots x_n$

### 1.2. 二进制转十进制

整数部分

逐次乘基相加法，从高位向低位

$(1011)_2 = (11)_{10}$

$1 \times 2+0=2$

$2 \times 2+1=5$

$5 \times 2+1=11$

小数部分

逐次除基相加法，从低位向高位

$(0.1011)_2=(0.685)_{10}$

$1/2=0.5$

$(1+0.5)/2=0.75$

$(0+0.75)/2=0.375$

$(1+0.375)/2=0.6875$
