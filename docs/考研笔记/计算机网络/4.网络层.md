## 1. 概述

### 1.1. 主要协议

网络层：IP, ICMP, IGMP, ARP, RARP

运输层：TCP, UDP

应用层：HTTP, FTP, SMTP等

### 1.2. 主要中间设备

根据中间设备所在的层次，可以有以下4种不同的中间设备：

1. 物理层使用的中间设备叫作转发器（repeater）；
2. 数据链路层使用的中间设备叫作网桥或桥接器（bridge）；
3. 网络层使用的中间设备叫作路由器（router）；
4. 在网络层以上使用的中间设备叫作网关（gateway），用网关连接两个不兼容的系统需要在高层进行协议的转换。
